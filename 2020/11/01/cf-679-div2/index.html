C - Perform Easily

Description

题目链接

Solution

对于每一个 bib_ibi​，将 <bi−a[j],i>(1≤j≤6)<b_i - a[j], i>(1 \le j \le 6)<bi​−a[j],i>(1≤j≤6) 求出并按第一关键字排序。

遍历排序后的二元组，对于当前二元组 iii，向右找到第一个 jjj 使得它们之间二元组的第二关键字有 nnn 种，那么这样的 iii、jjj 的第一关键字对应的区间可以被计入答案。

由于 iii 单增时 jjj 也是单增的，故可用双指针扫描。判断第二关键字是否有 nnn 种可用 map<int, int> cnt 获取 size() 来实现，当一个数的 cntcntcnt 变为 0 时，将其从 mapmapmap 中 erase 即可 (似乎只用数组就够了)。

D - Shurikens

Description

题目链接

Solution

按事件发生的相反顺序考虑。由于每次拿走的 shurikenshurikenshuriken 的价格是最小的，故遇到 ‘-’ 事件时，将对应价格插入 MinHeapMinHeapMinHeap；否则从 MinHeapMinHeapMinHeap 中弹出堆顶元素并压入栈，那么剩下的元素一定在这个事件发生之前就有了。

不合法情况：当前事件为 ‘+’ 且堆为空；或者当前事件为 ‘-’，对应的价格大于堆顶元素 (也就是拿走的价格不是最小的)。

最后输出栈中元素即可。
