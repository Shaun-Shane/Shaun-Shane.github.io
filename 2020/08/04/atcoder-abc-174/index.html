C Repsept

Description

求 7，77，777，7777，7，77，777，7777，7，77，777，7777， 中的第几个数是给定的 KKK 的倍数 (K≤ 106K \le\ 10^6K≤ 106)，没有则输出 −1-1−1。

Solution

  • 设 f(n)f(n)f(n) 为数列中第 n 个数的值，则有 f(n)=10∗f(n−1)+7f(n) = 10 * f(n - 1) + 7f(n)=10∗f(n−1)+7，两边同除以 10n10^n10n，求通项可得:

    f(n)=7∗(10n−1)/9f(n) = 7 * (10^n - 1) / 9 f(n)=7∗(10n−1)/9

  • 问题转化为求 f(n)% K=0f(n) \%\ K = 0f(n)% K=0 所对应的 nnn
  • 可以发现，KKK 是 2 或 5 的倍数时无解。有解时，一定有 n<Kn < Kn<K
  • 故可枚举 nnn，使用 mod=Kmod = Kmod=K 的快速幂判断即可

E Logs

Description

有 NNN 个木头，现需要将这些木头砍 KKK 次(设选取的木头长度为L，则砍了之后得到长度为 ttt (0<t<L0 < t < L0<t<L), L−tL - tL−t 的木头，砍了之后的木头还能继续砍)，求 KKK 次操作后这些木头中最大长度的最小值(向上取整)。

Solution

二分木头最大长度的最小值 midmidmid，并 checkcheckcheck 把所有木头分为长度 l≤ midl \le\ midl≤ mid 的木头所需要的次数，与 kkk 比较即可。
注意木头的最小长度应取 1 而不是 0。

F Range Set Query

Description

给定一个序列，多次询问区间 [li,ri][l_i, r_i][li​,ri​] 不同数字的个数。

Solution

  • 记录 aia_iai​ 前一次出现的位置 pre[ai]pre[a_i]pre[ai​](若 aia_iai​ 第一次出现则为 0)
  • 对于区间 [l,r][l, r][l,r]，如果区间的一个数 per[ai]≤lper[a_i] \le lper[ai​]≤l，则该数在区间内第一次出现，可计入答案
  • 于是以下标为值域，建立可持久化至于线段树。每个版本在 pre[ai]pre[a_i]pre[ai​] 位置权值加 1，最后统计 [0,li−1][0, l_i - 1][0,li​−1] 的权值总和即可
