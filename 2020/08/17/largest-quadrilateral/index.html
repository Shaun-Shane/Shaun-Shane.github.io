SCOI2007 最大土地面积

  • 题目链接
  • 计算几何/凸包/旋转卡壳

Description

选取平面上的 4 个点，使得围成的四边形的面积最大 (n≤2000n \le 2000n≤2000)。

Solution

最大四边形的顶点一定在凸包上。
当凸包顶点数大于 3 时，先选取一条对角线，则其它两个点到对角线的距离具有单峰性，套旋转卡壳即可求出最大四边形面积。
当凸包顶点数等于 3 时，最大四边形面积为凸包面积减去内部点与凸包上的点构成的最小的三角形面积。
当凸包顶点数等于 2 时，面积为 0。
