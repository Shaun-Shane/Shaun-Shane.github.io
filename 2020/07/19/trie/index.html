字典树笔记

发表于2020-07-19更新于2023-09-17字数统计14.1k阅读时长146分作者Shane

Trie，即字典树，是一个非常强大的数据结构，可以高效处理字符串、异或最大值以及数集维护问题，而且常数非常小。

下面的题可用用 Trie 树实现，主要分为4个等级。

Level 1

Trie 树基础

Description

P2580 于是他错误的点名开始了

Solution

首先把给出的 nnn 个字符串插入 trietrietrie 树中。之后询问时，如果给出的字符串在 trietrietrie 树中不存在，否则在字符串末尾对应节点打上标记，如果这个点之前已经被打过标记，说明这个字符串在之前已经被询问过了。

Description

SP4033 PHONELST - Phone List

Solution

考虑两种情况，若当前串是前面已插入的串中某一个串的前缀，那么它在 $ trie$ 上对应的最后一个节点之前应该被访问过；若前面已插入的串中存在当前串的前缀，那么在插入当前串时，一定会访问到 trietrietrie 上的某个节点，且该节点对应之前某个串的结尾。因此只需要打两个标记，一个记录节点是否被访问过，一个记录节点是否对应某个串的结尾，这两个标记边插入边更新即可。

A Xor Sum

Description

HDU 4825 - Xor Sum

Solution

首先将集合中的每个数化为二进制形式，分别从高位到低位依次插入 trietrietrie 树中。对于任意给定的数，要使异或结果最大，那么异或结果的每一位尽可能是1，也就是说给定的数，和使异或结果最大的数，在每一位的值尽可能相反。每次贪心的选取，在 trietrietrie 树上走互补的链即可。单次查询的复杂度为
log2(S)log_2(S)log2​(S)。

考虑如何求 l1,2l_{1,2}l1,2​ 和 r1,2r_{1,2}r1,2​。将 a1a_1a1​ 到 ana_nan​ 按从大到小排序，同时记录原来的下标 bib_ibi​。排序后按顺序将 bib_ibi​ 插入 setsetset 中。易知 setsetset 中任意元素 bkb_kbk​ 对应值 abka_{b_k}abk​​一定大于当前位置 bib_ibi​ 对应值 abia_{b_i}abi​​(题目约定数组 aaa 中的元素不重复), 且 setsetset 已按下标排好序，用 lower_bound 和 upper_bound 容易求出当前下标 bib_ibi​ 的前驱和后继。
