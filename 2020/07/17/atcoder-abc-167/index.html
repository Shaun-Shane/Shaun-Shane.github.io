D Teleporter

Description

有 nnn 个小镇，小镇 iii 有一个传送能传送到小镇 jjj。从小镇 1 开始传送，求出经过 kkk 次传送最终到哪个小镇。

Solution

可通过拓扑排序找到环(拓扑排序后入度 ininin 不为 0 的节点为环上的节点)。然后处理出环长，起点到环的距离，即可判断传送 kkk 次后所在的节点编号。

E Colorful Blocks

Description

有排列于一行的 NNN 个格子，每个格子染上颜色 1 到 MMM，求至多有 KKK 对颜色相同的相邻格子的染色方案数。

Solution

  • KKK 对相邻颜色相同，则相邻颜色相同的一对格子缩成一个格子，则有 N−KN−KN−K 个格子颜色互不相同，其中一个格子颜色有 MMM 种取法，其它格子只有 M−1M−1M−1 种。
  • 所以答案为 ΣCn−1im(m−1)n−i−1\Sigma C_{n-1}^im(m−1)^{n−i−1}ΣCn−1i​m(m−1)n−i−1

F Bracket Sequencing

Description

给出n个括号序列，总长不超过 10610^6106，能否用它们按照一定顺序拼接成一个合法的括号序列。

Solution

  • 设 “(“ 的权为 1，“)” 的权为 -1。
  • 对于一个合法括号序列，任意前缀和一定大于等于 0，且总权值和为 0(任意位置只允许有 “(“ 待配对，且整个序列 “(“ 和 “)” 数量相同)
  • 首先用 sumsumsum 记录 nnn 个序列中权值的和，sumsumsum 不为 0 一定不能拼接出合法序列
  • 考虑一种最优的拼接方案，如果该方案不成功，则其他方案一定不会成功
    • 显然，先选权值和大于等于零的，且任意前缀最小值大的先选
    • 接下来取权值和小于零的。如果用前面的选取方法，可以举出反例
  • 接下来推权值和小于零时一种方案更优的等价条件
    • 设 iii 和 jjj 是之后紧接着拼接的序列，它们之前已构造和为 sumsumsum 的序列，那么这两个序列放入后满足任意前缀大于等于 0
    • 先 iii 后 jjj，则 sum+cnti+min⁡prej>=0sum+cnt_i+\min pre_j>=0sum+cnti​+minprej​>=0
    • 先 jjj 后 iii，则 sum+cntj+min⁡prei>=0sum+cnt_j+\min pre_i>=0sum+cntj​+minprei​>=0
    • 设先放 iii 更优，有 cnti−min⁡prei>cntj−min⁡prejcnt_i−\min pre_i >cnt_j−\min pre_jcnti​−minprei​>cntj​−minprej​
