国家集训队 种树

  • 题目链接
  • 数据结构/优先队列/链表
  • STL/list
  • 贪心

Description

有一个环形广场上有 nnn 个种树位置，顺时针编号 1 到 nnn。每个位置有美观度 AiA_iAi​，选取不相邻的 mmm 个位置种树，求最大美观度和。

Solution

思路同APIO/CTSC 2007 数据备份，只不过需通过环形链表解决，且当 n/2<mn / 2 < mn/2<m 时无解。

Code

环形双向链表

与非环形双向链表不同，headheadhead 与 tailtailtail 不用作为虚拟节点。链表建立时 tail−>next=headtail->next = headtail−>next=head 且 head−>prev=tailhead->prev = tailhead−>prev=tail。删除和插入操作与非双向链表类似。

STL 环形链表

STLSTLSTL 的 listlistlist 是环形链表。list.end() 的下一个迭代器是 list.begin()，list.begin() 的上一个迭代器是 list.end()。删除过程中，需特判当前迭代器是否为 list.end()，若是，则选取下一个迭代器进行删除。另外需注意：由于最后一次删除操作可能导致 listlistlist 为空，要特判一下。
