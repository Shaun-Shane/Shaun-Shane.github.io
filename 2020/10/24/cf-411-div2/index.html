D - Minimum number of steps

Description

将只含 aaa、bbb的字符串中的 ababab 全部替换为 bbabbabba 需要多少次操作

Solution

字符串最终会变为 bbbb....aaa....bbbb....aaa....bbbb....aaa....。将 ababab 替换为 bbabbabba 后 aaa 的个数不变，前面的结果对后面没有影响，所以 bbb 前有多少 aaa 就要操作 2numa−12^{num_a} - 12numa​−1 次。

F - Expected diameter of a tree

Description

给出一个森林，m 次询问，每次给出节点 aaa, bbb，将 aaa 所在的树与 bbb 所在的树连一条边，求连接后的树的直径的期望(每次操作相互独立)。

Solution

先预处理出每个点属于哪一棵树，每棵树的 sizesizesize、直径，以及树上每个点到最远点的距离 (可以证明，树上一点的最远点一定是树的直径的端点)。

可以二次 dfsdfsdfs 求出树的直径和端点，再从端点出发，求出到每一点的最远距离 dis[]dis[]dis[]。

然后枚举 aaa 所在树的点 iii，二分 bbb 所在树的点 jjj，求出 disi+disj+1>max(diameterbelongi,diameterbelongj)dis_i + dis_j + 1 > max(diameter_{belong_i}, diameter_{belong_j})disi​+disj​+1>max(diameterbelongi​​,diameterbelongj​​) (需要提前将 dis[]dis[]dis[] 排序)。

然后小于 jjj 的部分，树的直径都是 max(diameterbelongi,diameterbelongj)max(diameter_{belong_i}, diameter_{belong_j})max(diameterbelongi​​,diameterbelongj​​)，剩余部分的直径和通过前缀和求出 (前缀和需要 dis[]dis[]dis[] 排序后预处理)。
