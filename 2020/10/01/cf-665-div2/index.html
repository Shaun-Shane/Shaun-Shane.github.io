B Ternary Sequence

考虑 aaa 序列，aaa 的 2 优先和 bbb 的 1 匹配，若有剩余再和 bbb 的 2 匹配。然后 2 和 0， 0 和 2 匹配。最后只能 2 和 1 匹配。

C Mere Array

Solution

设最小值为 mimimi，则一定可以通过 mimimi 让 mimimi 的倍数排成有序的。把 mimimi 的倍数先取出排序，再放回原序列，再判断整个序列是否升序。

D Maximum Distributed Tree

Solution

首先对于一条边，它会被计算两端的子树大小的乘积次。要让答案最大，那么被计算次数最多的边的边权应为最大的约数，依此类推。注意如果质因数个数超过 n−1n - 1n−1，则将最大的几个质因子乘起来，分配给被计算最多的边。

E Divide Square

Solution

有两种情况会使答案加一：贯穿整个矩形的直线，或两条直线相交。

求直线交点：可以将水平线段按左端点排序，左端点相同按右端点排序。并将竖直线段的 xxx 坐标也加入，若 xxx 坐标与水平线段左端点相同，优先考虑水平线段。

统计交点数：可以通过树状数组对 yyy 进行区间 [yl,yr][y_l, y_r][yl​,yr​] 查询，加入或删除水平线则对水平线的 yyy 单点修改。

加入或删除水平线可通过队列实现，如果右端点小于当前竖直线的 xxx，则出队，并通过树状数组对 yyy 进行单点修改。

注意：树状数组不是 0 based 的，需将坐标都加 1。

F Reverse and Swap

Solution

考虑线段树维护，设线段树从上往下是第 0 层，第 1 层，第 2 层，…，第 n 层。那么 swap(k)swap(k)swap(k) 操作就是给第 n−k+1n - k + 1n−k+1 层所有节点打上交换标记，reverse(k)reverse(k)reverse(k) 操作就是给第 n−kn - kn−k 到 nnn 层的所有节点打上标记。

一种做法是用 tag[]tag[]tag[] 记录每一层的交换标记，然后每个节点也有一个 tagtagtag 标记，如果节点的 tagtagtag 和节点对应层的 tag[depth]tag[depth]tag[depth] 不同，则交换其左右儿子，并修改 tag。
注：当区间翻转的是一个整区间时才能用线段树交换左右儿子，其次结构体中不建议保存节点 l,rl, rl,r 信息（不方便交换左右儿子，因为左右儿子的 l,rl, rl,r 都要变），可以直接作为函数参数。

另一种做法是根据层标记确定往左二子还是右儿子走，同时计算变化后的区间。
