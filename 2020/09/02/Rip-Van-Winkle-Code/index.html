Rip Van Winkle’s Code

  • 题目链接
  • 线段树

Description

有一个序列初始值全为零，现有一下四种操作：

  • A l r 将区间 [l,r][l, r][l,r] 加上等差数列 1,2,...,r−l+11, 2, ..., r - l + 11,2,...,r−l+1
  • B l r 将区间 [l,r][l, r][l,r] 加上等差数列 r−l+1,r−l,...,1r - l + 1, r - l, ..., 1r−l+1,r−l,...,1
  • C l r x 将区间 [l,r][l, r][l,r] 赋值 xxx
  • S l r 求区间 [l,r][l, r][l,r] 的和

Solution

难点在于区间加等差数列。
用 LvalLvalLval 表示表示该区间增加的等差数列的首项标记，tag2tag2tag2 表示表示该区间增加的等差数列的公差标记。因为一个加上等差数列的区间依旧是一个等差数列区间，于是两个标记可以累加。左右儿子增加的 sumsumsum 由父亲节点的 LvalLvalLval 和 tag2tag2tag2 推出。
注意：LvalLvalLval 和 tag2tag2tag2 都为 0 时标记不用下放；当有区间赋值标记时，等差数列标记无效，赋为 0。
