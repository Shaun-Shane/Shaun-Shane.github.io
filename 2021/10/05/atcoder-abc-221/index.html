Description

题目链接

Solution

首先求出树的直径，设为 DDD，直径上的点依次记为 V0,V1,…,VDV_0,V_1,\dots,V_DV0​,V1​,…,VD​。下面对直径分奇偶讨论：

  1. D 为奇数
    设 A=VD−12A=V_{\frac{D-1}{2}}A=V2D−1​​, B=VD+12B = V_{\frac{D+1}{2}}B=V2D+1​​。割掉连接 A,BA,BA,B 的边，得到两个子树 TA,TBT_A,T_BTA​,TB​。易证 TAT_ATA​ 中任意两点距离不超过 D−1D-1D−1，TBT_BTB​ 中任意两点距离不超过 D−1D-1D−1，因此距离为 DDD 的两点只能一个在 TAT_ATA​ 中，一个在 TBT_BTB​ 中。设 TAT_ATA​ 中和 AAA 距离为 D−12\frac{D-1}{2}2D−1​ 的点有 M1M_1M1​ 个，TBT_BTB​ 中和 BBB 距离为 D−12\frac{D-1}{2}2D−1​ 的点有 M2M_2M2​ 个，那么答案为 M1M2M_1M_2M1​M2​。
  2. D 为偶数
    设 C=VD2C=V_{\frac{D}{2}}C=V2D​​。将 CCC 以及它所连的边（设有 kkk) 条删去，得到 kkk 个子树，记为 T1,T2,…,TkT_1,T_2,\dots,T_kT1​,T2​,…,Tk​，它们的根为 R1,R2,…,RkR_1,R_2,\dots,R_kR1​,R2​,…,Rk​。易证 TiT_iTi​ 中任意两点距离不超过 D−2D-2D−2。因此符合条件的点集中，任意两点不在同一子树中。设 TiT_iTi​ 中和 RiR_iRi​ 距离为 D2−1\frac{D}{2}-12D​−1 的点有 MiM_iMi​ 个，那么答案为 (M1+1)(M2+1)…(Mk+1)−(M1+M2+⋯+Mk)−1(M_1+1)(M_2+1)\dots(M_k+1)-(M_1+M_2+\dots+M_k)-1(M1​+1)(M2​+1)…(Mk​+1)−(M1​+M2​+⋯+Mk​)−1。

G Jumping sequence

Description

题目链接

Solution

将坐标系顺时针旋转 45°45°45°，再施加 2\sqrt{2}2​ 的缩放，得到新的坐标系 (X+Y,X−Y)(X+Y,X-Y)(X+Y,X−Y)。在这个坐标系中，向上移动表示为 (+D,−D)(+D,-D)(+D,−D)，向下移动表示为 (−D,+D)(-D,+D)(−D,+D)，向左移动表示为 (−D,−D)(-D,-D)(−D,−D)，向右移动表示为 (+D,+D)(+D,+D)(+D,+D)，从而问题转化为找到两个 +1,−1+1,-1+1,−1 序列 s,s′s,s's,s′，使得

∑i=1NsiDi=A+B,∑i=1Nsi′Di=A−B\sum\limits_{i=1}^{N} s_iD_i=A+B,\sum\limits_{i=1}^{N} s'_iD_i=A-B i=1∑N​si​Di​=A+B,i=1∑N​si′​Di​=A−B

定义 S=∑i=1NDiS=\sum\limits_{i=1}^{N}D_iS=i=1∑N​Di​，那么问题可以继续转化为找到 0,10,10,1 序列 t,t′t,t't,t′，满足

∑i=1NtiDi=S+A+B2,∑i=1Nti′Di=S+A−B2\sum\limits_{i=1}^{N} t_iD_i=\frac{S+A+B}{2},\sum\limits_{i=1}^{N} t'_iD_i=\frac{S+A-B}{2} i=1∑N​ti​Di​=2S+A+B​,i=1∑N​ti′​Di​=2S+A−B​

显然上述等式右边小于零，大于 SSS，或者不是偶数时无解。否则，设 dp[i][s]dp[i][s]dp[i][s] 表示考虑前 iii 个数，和为 sss 的方案是否存在，我们可以在 O(NS)O(NS)O(NS) 时间内求解。由于 NSNSNS 可达 7.2∗1097.2*10^97.2∗109，朴素 DPDPDP 无法通过此题。考虑到我们只关心方案是否存在，因此可用 bitsetbitsetbitset 优化。

判断完可行性后，从终态倒着往前推，枚举四个方向，看之前的状态是否可行来确定每一步的方向。

H Count Multiset

Description

题目链接

Solution

将 multiset 中元素从小到大排列，记为 A1,A2,…,AkA_1,A_2,\dots,A_kA1​,A2​,…,Ak​。

令 B1=A1B_1 = A_1B1​=A1​，Bi=Ai−Ai−1,i≥2B_i = A_i-A_{i-1},i\ge 2Bi​=Ai​−Ai−1​,i≥2，那么问题转化为：

对 k=1,2,…,Nk=1,2,\dots,Nk=1,2,…,N，求满足下面三个条件的序列个数：

  • B1>0B_1>0B1​>0。
  • 不存在长度超过 M−1M-1M−1 的连续的 000。
  • ∑i=1kBi∗(k−i+1)=N\sum\limits_{i=1}^{k}B_i*(k-i+1)=Ni=1∑k​Bi​∗(k−i+1)=N。

将 BBB 翻转，那么上述问题可以继续转化为：

对 k=1,2,…,Nk=1,2,\dots,Nk=1,2,…,N，求满足下面三个条件的序列个数：

  • Bk>0B_k>0Bk​>0。
  • 不存在长度超过 M−1M-1M−1 的连续的 000。
  • ∑i=1kBi∗i=N\sum\limits_{i=1}^{k}B_i*i=Ni=1∑k​Bi​∗i=N。

设 dp[i][s]dp[i][s]dp[i][s] 表示求满足下列条件的 BBB 的个数：

  • Bi>0B_i>0Bi​>0
  • 不存在长度超过 M−1M-1M−1 的连续的 000。
  • ∑j=1iBj∗j=N\sum\limits_{j=1}^{i}B_j*j=Nj=1∑i​Bj​∗j=N。

易得状态转移方程：dp[i][s]=∑j=i−M+1i−1∑k=1⌊si⌋dp[j][s−i∗k]dp[i][s] = \sum\limits_{j=i-M+1}^{i-1}\sum\limits_{k=1}^{\lfloor \frac{s}{i} \rfloor}dp[j][s-i*k]dp[i][s]=j=i−M+1∑i−1​k=1∑⌊is​⌋​dp[j][s−i∗k]。直接转移需要枚举 i,j,k,si,j,k,si,j,k,s，时间复杂度 O(N3M)O(N^3M)O(N3M) 无法通过此题。

对状态转移式进行变形，得 dp[i][s]=∑k=1⌊si⌋∑j=i−M+1i−1dp[j][s−i∗k]=∑k=1⌊si⌋(∑j=1i−1dp[j][s−i∗k]−∑j=1i−M+1dp[j][s−i∗k])dp[i][s] = \sum\limits_{k=1}^{\lfloor \frac{s}{i} \rfloor}\sum\limits_{j=i-M+1}^{i-1}dp[j][s-i*k] = \sum\limits_{k=1}^{\lfloor \frac{s}{i} \rfloor}(\sum\limits_{j=1}^{i-1}dp[j][s-i*k]-\sum\limits_{j=1}^{i-M+1}dp[j][s-i*k])dp[i][s]=k=1∑⌊is​⌋​j=i−M+1∑i−1​dp[j][s−i∗k]=k=1∑⌊is​⌋​(j=1∑i−1​dp[j][s−i∗k]−j=1∑i−M+1​dp[j][s−i∗k])。我们先按第一维求和，再对第二维以间隔 iii 求和即可在 O(N2)O(N^2)O(N2) 时间内完成求解。
