D Staircase Sequences

Description

题目链接

Solution

设首项为 aaa，项数为 nnn，那么有 n∗(2a+n−1)=2Nn*(2a+n-1)=2Nn∗(2a+n−1)=2N。由于 nnn 为正整数，aaa 为整数，因此 nnn 为 2N2N2N 的因数，且 2Nn−n+1\frac{2N}{n}-n+1n2N​−n+1 是偶数。不同的 nnn 对应不同的数列，因此答案为满足上述条件的 nnn 的个数。

E Magical Ornament

Description

题目链接

Solution

称必须包括的 KKK 种宝石为关键宝石，本题实际上是求对这些宝石进行排列的代价。我们只需要考虑关键宝石两两之间间接相连中间至少要通过几个宝石，之后便容易用状压 dpdpdp 求解。具体的，我们分别以这 KKK 个关键宝石为起点跑 KKK 次最短路，同时记录从起点到其余关键宝石的最短距离，并用数组 disdisdis 记录。设 dp[i][j]dp[i][j]dp[i][j] 表示当前宝石集合为 iii，最后一个加入的为第 jjj 个关键宝石所需的最少宝石数目，那么有 dp[i][j]=dp[i⊕(1<<j)][k]+dis[k][j]dp[i][j] = dp[i⊕(1<<j)][k] +dis[k][j]dp[i][j]=dp[i⊕(1<<j)][k]+dis[k][j]，于是答案即为 min⁡1≤i≤Kdp[(1<<K)−1][i]\min \limits_{1 \le i \le K}dp[(1<<K)-1][i]1≤i≤Kmin​dp[(1<<K)−1][i]。

上述做法时间复杂度为 O(K(N+M)+K22K))O(K(N+M)+K^22^K))O(K(N+M)+K22K))。
