Solution

对于 (i,j)(i,j)(i,j)，我们只需要考虑 i′≤ii' \le ii′≤i 的点和它之间的代价，分两种情况：

  1. j′≤jj' \le jj′≤j，代价为 Ai,j+C∗(i+j)+Ai′,j′−C∗(i′+j′)A_{i,j}+C*(i+j)+A_{i',j'}-C*(i'+j')Ai,j​+C∗(i+j)+Ai′,j′​−C∗(i′+j′)。

Description

题目链接

Solution

考虑 KruskalKruskalKruskal 算法，每次选择最小的不形成环的边不断加入。然而其时间复杂度为 O(ElogE)O(ElogE)O(ElogE)，其中 EEE 为图的边数。

在本题中，图的边数是巨大的，所以需要加快 KruskalKruskalKruskal 算法加边的过程。

设从小到大枚举到 CtC_tCt​ 时 KruskalKruskalKruskal 算法形成的图为 GtG_tGt​，XtX_tXt​ 为 GtG_tGt​ 中联通块的个数。则新增的联通块个数为 Xt−1−XtX_{t-1} - X_tXt−1​−Xt​。

所以最终答案为 ∑t=1MCt(Xt−1−Xt)\sum\limits_{t=1}^{M}C_t(X_{t-1}-X_t)t=1∑M​Ct​(Xt−1​−Xt​)。如果 XM>1X_M > 1XM​>1，则最终图不连通，输出 -1。

如何求得 XtX_tXt​：
因为对于任意 i=1,2,...,ti = 1, 2, ..., ti=1,2,...,t 和任意 x=0,1,...,N−1x = 0, 1, ..., N - 1x=0,1,...,N−1，我们可以从顶点 xxx 移动到 (x+Ai)mod  N(x + A_i) \mod N(x+Ai​)modN。所以在 GtG_tGt​ 中，我们可以从 vvv 移动到 www 当且仅当存在整数 k1,k2,...,ktk_1, k_2, ..., k_tk1​,k2​,...,kt​，使得 w≡v+k1A1+k2A2+...+ktAtmod  Nw \equiv v + k_1A_1+k_2A_2+...+k_tA_t \mod Nw≡v+k1​A1​+k2​A2​+...+kt​At​modN

上述式子 ⇔\Leftrightarrow⇔ w=v+k0N+k1A1+k2A2+...+ktAtw = v + k_0N+k_1A_1+k_2A_2+...+k_tA_tw=v+k0​N+k1​A1​+k2​A2​+...+kt​At​ ⇔\Leftrightarrow⇔ w=v+kdtw=v + kd_tw=v+kdt​ ⇔\Leftrightarrow⇔ w≡vmod  dtw \equiv v \mod d_tw≡vmoddt​

F Coprime Solitaire

Description

题目链接

Solution

枚举 222 到 2∗1062*10^62∗106 的素数 ppp。

对于每个 ppp，设所选择的卡牌为 Xp,1,Xp,2,...,Xp,kpX_{p,1}, X_{p, 2}, ..., X_{p,k_p}Xp,1​,Xp,2​,...,Xp,kp​​ (区分正反面, 即 Xp,iX_{p,i}Xp,i​ 可能是某个卡牌的正面或反面)。

则 Takahashi is happy 当且仅当 ∀p(≤2∗106)\forall p(\le 2*10^6)∀p(≤2∗106) 且 ppp 为素数，卡牌朝上的数字中只有至多一个有质因子 ppp。即选了 Xp,iX_{p,i}Xp,i​，其它 Xp,jX_{p,j}Xp,j​ 都不能选。

转化为逻辑表达式为 F:=⋀p:prime,p≤2∗106⋀1≤i,j≤kp,i≠j(Xp,i→¬Xp,j)=trueF:=\bigwedge\limits_{p:prime,p\le 2*10^6}\bigwedge\limits_{1\le i,j \le k_p, i \ne j} (X_{p,i} \rightarrow \neg X_{p,j}) = trueF:=p:prime,p≤2∗106⋀​1≤i,j≤kp​,i​=j⋀​(Xp,i​→¬Xp,j​)=true

我们令 Fp:=⋀1≤i,j≤kp,i≠j(Xp,i→¬Xp,j)F_p:= \bigwedge\limits_{1\le i,j \le k_p, i \ne j} (X_{p,i} \rightarrow \neg X_{p,j})Fp​:=1≤i,j≤kp​,i​=j⋀​(Xp,i​→¬Xp,j​)

该问题可通过 2-SAT 算法解决。但是根据上述表达式建图边数过多，无法在规定时间内通过该题，还需要引入 2 类增补变量 Yp,1,Yp,2,...,Yp,kpY_{p,1}, Y_{p,2},...,Y_{p,k_p}Yp,1​,Yp,2​,...,Yp,kp​​ 和 Zp,1,Zp,2,...,Zp,kpZ_{p,1}, Z_{p,2},...,Z_{p,k_p}Zp,1​,Zp,2​,...,Zp,kp​​，用 Fp′F_p'Fp′​ 来替换 FpF_pFp​:

Fp′:=⋀1≤i≤kp(Yp,i→¬Xp,i)∧⋀2≤i≤kp(Yp,i→Yp,i−1)∧⋀2≤i≤kp(Xp,i→Yp,i−1)∧⋀1≤i≤kp(Zp,i→¬Xp,i)∧⋀1≤i≤kp−1(Zp,i→Zp,i+1)∧⋀1≤i≤kp−1(Xp,i→Zp,i+1)F_p':= \bigwedge\limits_{1\le i\le k_p}(Y_{p,i} \rightarrow \neg X_{p,i}) \wedge \bigwedge\limits_{2\le i\le k_p}(Y_{p,i} \rightarrow Y_{p,i-1}) \wedge \bigwedge\limits_{2\le i\le k_p}(X_{p,i} \rightarrow Y_{p,i-1}) \wedge \\ \bigwedge\limits_{1\le i\le k_p}(Z_{p,i} \rightarrow \neg X_{p,i}) \wedge \bigwedge\limits_{1\le i\le k_p-1}(Z_{p,i} \rightarrow Z_{p,i+1}) \wedge \bigwedge\limits_{1\le i\le k_p-1}(X_{p,i} \rightarrow Z_{p,i+1}) Fp′​:=1≤i≤kp​⋀​(Yp,i​→¬Xp,i​)∧2≤i≤kp​⋀​(Yp,i​→Yp,i−1​)∧2≤i≤kp​⋀​(Xp,i​→Yp,i−1​)∧1≤i≤kp​⋀​(Zp,i​→¬Xp,i​)∧1≤i≤kp​−1⋀​(Zp,i​→Zp,i+1​)∧1≤i≤kp​−1⋀​(Xp,i​→Zp,i+1​)

画下图可知这些增补变量相当于两条链，把 Xp,iX_{p,i}Xp,i​ 与 Xp,1≤j<iX_{p,1\le j < i}Xp,1≤j<i​ 以及 Xp,iX_{p,i}Xp,i​ 与 Xp,i<j≤kpX_{p, i<j \le k_p}Xp,i<j≤kp​​ 间接连接起来，从而不用两两加边。

最后令 F′=⋀p:prime,p≤2∗106Fp′F'=\bigwedge\limits_{p:prime,p\le 2*10^6}F_p'F′=p:prime,p≤2∗106⋀​Fp′​，则 F′⇔FF' \Leftrightarrow FF′⇔F。
